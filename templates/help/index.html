Getting Started

{{ config.NAME }} is a knowledge base for space history, mostly regarding starship. It allows users to create and share information easily. it also supports branches, which makes organizing related content simple and efficient.

  • To create your first page, start by searching for the title you’d like to use. If no results appear, you’ll have the option to create a new page with that title.
  • Use Markdown syntax to format your content
  • Link pages together using double brackets: [[ Page Name:branch ]]

Editing Pages

Editing pages is simple and intuitive. Click the edit button on any page to begin making changes.

  • Use headings (##, ###) to structure your content
  • Insert images using ![](image-url)
  • Use code blocks with triple backticks (```) for code snippets

Navigation

{{ config.NAME }} provides multiple ways to navigate your content.

  • Use the sidebar to browse your pages
  • Search for pages using the search bar at the top
  • Use breadcrumbs to trace your path through the wiki

Permissions

You can only edit if you're logged in.

  • Public pages are visible to everyone
  • Accounts only Username and Password, No email verifications/phone numbers
  • Private pages are only visible to authorized users
